# stream-example

Full example to showcase Stream features

**Features:**
- Room with chat and video

**Future features:**
- Starting private rooms


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
